# Firefox Custom Browser Project Structure

- source/: Firefox ESR source code will be placed here
- patches/: Custom patches for de-googling, privacy, and Proton integration
- build/: Compiled browser output and .deb packages
- scripts/: Automation scripts for building, patching, and packaging
- docs/: Documentation and guides

## Next Steps
1. Download Firefox ESR source into `source/`
2. Apply patches from `patches/`
3. Build in `build/`
4. Use scripts in `scripts/` to automate tasks
5. Document changes in `docs/`
